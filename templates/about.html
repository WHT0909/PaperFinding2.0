关于 PaperFinding 2.0

什么是 PaperFinding?

PaperFinding 是一个基于PubMed的学术文献检索系统，旨在帮助生物医学领域的研究人员快速查找所需的学术论文。 我们提供简洁直观的界面，让您轻松访问 PubMed 数据库中的海量文献资源。

PaperFinding 能帮你做什么？

本系统使用 PubMed 数据库作为文献来源。PubMed 是美国国家医学图书馆(NLM)维护的生物医学文献数据库， 包含超过 3500 万篇生物医学文献，涵盖了生命科学领域的广泛研究成果。

技术架构

  • 后端: FastAPI (Python) - 高性能 Web 框架
  • 前端: HTML5 + CSS3 + JavaScript - 响应式设计
  • API: NCBI Entrez E-utilities - 官方 PubMed 接口
  • 数据库: PubMed - 全球最大的生物医学文献数据库

功能特点

  • 🔍 关键词快速检索 - 支持布尔运算符和高级搜索
  • 📄 分页浏览结果 - 每页显示 10 篇文献，支持翻页
  • 📊 时间范围筛选 - 可选择近 5 年、10 年或更早文献
  • 📝 显示文献详细信息（标题、作者、期刊、发表日期）
  • 📋 提供文献摘要 - 帮助快速了解文献内容
  • 🔗 DOI链接跳转 - 直接访问原文来源

使用说明

1. 在搜索框中输入您感兴趣的研究主题或关键词
2. 选择时间范围（可选）
3. 点击搜索按钮或按Enter键进行检索
4. 浏览搜索结果，点击文献标题查看详细信息
5. 使用底部分页按钮浏览更多结果

注意事项

- 本系统仅提供文献检索服务，不存储或复制任何文献内容
- 文献版权归原作者和期刊所有
- 建议使用稳定的网络连接以获得最佳体验
- 如需获取全文，请通过DOI链接访问原文
